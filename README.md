Digit Manipulation & Search (C++)
Цей проєкт реалізує алгоритм пошуку найближчого більшого або рівного трицифрового числа, що складається з однакових цифр. Робота демонструє навички декомпозиції чисел та використання циклів для оптимізації пошуку.

Технології:
C++

Оператори ділення (%) та залишку (/) — використані для розбиття числа на окремі розряди (одиниці, десятки, сотні).

Бібліотека <windows.h> — для забезпечення коректної роботи з кириличними шрифтами в інтерфейсі консолі.

Що робить ця програма:
Аналізує структуру числа: Програма приймає трицифрове число та виділяє його розряди для подальших розрахунків.

Інтелектуальний пошук: За допомогою циклу for алгоритм перебирає можливі комбінації однакових цифр (111, 222, 333...), починаючи з першої цифри введеного числа.

Логіка переривання: Використано оператор break для миттєвої зупинки циклу при досягненні умови, що дозволяє оптимізувати виконання програми.

Обробка граничних значень: Програма коректно працює з числами до 999 включно.
